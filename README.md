# botUordiArti
bot para telegram que volta uma imagem em wordart

Dar **aquele** *npm install*:
```
npm install
```

Criar um arquivo chamado `environment.js` que exporta seu token:
``` javascript
/* environment.js */
module.exports = <SEU BOT API TOKEN AQUI>
```

Inicializar o bot:
```
node bot.js
```
